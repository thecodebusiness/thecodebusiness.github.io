Snesh : A unique way of buying and selling stuffs. Here you need to earn it to own it.

The Problem Statement

Sarvesh had a very nice idea of selling unique items online and he wanted to go live quickly before someone else come up with the similar idea.

Snesh, is the online eCommerce like auction site where user can bid on the items and as soon as auction event goes off then the automatically last bidder becomes the winner.

Winner should pay the bid amount online to confirm the ownership. If user fail to do so within the given time frame, product will be again available for auction.

Solution

TheCodeBusiness team worked closely with Sarvesh to conceptualise the idea on paper then in mockup designs.

Since, snesh is targeting the Indian audience we had to be prepared for the system's availability and fault tolerence. We at TheCodeBusiness decided to go ahead with the Django and Python. We quickly prototyped the system using Agile Development. We involved Sarvesh in each and every module release cycle to make sure that everything is going according to the plan.

Once the Sarvesh confirmed that system is designed and developed as per the plan then we turned the prototype into the full fledge system.

Below are the list of stacks which we used for Snesh.

Django
We mostly prefer the Django for any product development to fulfill the client's need. We designed CMS for auction and order management.

PostgreSQL
We used PostgreSQL to store the data and other informations like, hosted events, activity tracking etc.

Socket
We used socket programming to give a life to auction system so that it looks real when multiple people are bidding on the same item.

Goolge Cloud Platform
Google cloud platform is relatively cheaper than the AWS and easy to mange and host services.

SendGrid
SendGrid email service comes with very nice API integration with Django. We designed the templates and emailer service in django to send emails.

SMSGatewayHub
To send sms through the auction platform.

Tech Specs

Python Django PostgreSQL Google Cloud Storage
