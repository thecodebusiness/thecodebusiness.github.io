Maharashtra Property
Buy and Sell Property in Mira Road.

The Problem Statement

Manoj, Owner of the Maharashtra Property and Finance Consultant called TheCodeBusiness for a RealEstate website.

Website should look good and responsive. Just want to add top 10 properties on the website.

Everytime he goes to MagicBricks and 99Acres to add property details which is quite repetitive task. So he doesn't want to use another CMS to add same data on website.

Low budget.

Solution

Interesting. We asked Manoj to give us the top 10 property link [ from magicbricks ] which he wants to be listed on the website.

We came with a webcrawler solution which fetches the information from provided link and prepares a webpage with property information.

Manoj changes those top 10 links very frequently. Everyday during midnight this web-crawler runs and fetches the latest information from the link and updates the website.

Below are the list of stacks which are used for Maharashtra Property.


BeautifulSoup
Python based web-scrapping library.

Flask
Basic UI to add the link and webpage hosting.

Tech Specs
